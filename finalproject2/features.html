Overskrift

Samlepunkt er utviklet for å finne arrangementer, få nye vennekretser og for å følge med på ting som skjer rundt deg. I hverdagen bombarderes vi med hundrevis av arrangementer som bare ender opp med å distansere oss fra det. Derfor er appen lagd for å finne arrangementer som er skreddersydd for deg. For å gjøre dette mulig

Egen profil og grupper

I appen har man en egen profil, som fungerer uten for mye personlig informasjon. Om man vil kan man gjøre profilen sin public som gjør at andre kan se arrangementer som vedkommende skal på, tags som abonneres og valgt Geo-område. Ved private modus er ingenting gjort kjent via profil. Man lage og delta, på grupper som er offentlig eller privat. Gruppen kan også inneholde spesifikke tags.

Arrangementer skal kunne gjøres enten offentlig eller privat, og arrangøren kan velge mellom diverse innstillinger, blant annet en grense på nøyaktig hvor mange som får lov å bli invitert, om det er kun arrangør som kan invitere og tags.

Det skal kunne være mulig for arrangørene å sende kunngjøringer eller varsler angående oppdateringer eller endringer angående et arrangement. Da skal appen kunne sende en push-notification eller en mail til brukerne som er meldt på eller har vist interesse for arrangementet.

Tags og Geo-område

Den største delen av appen vil basere seg for tags innenfor angitt område. Det vil være generelle tags som man kan abonnere på som sport eller musikk. Det vil finnes tags som er mer spesifikke som basket eller konserter. Til slutt kan man være så spesifikke hvor man lager en egen tag for basketballaget eller bandnavnet sitt. Man vil kunne overstyre slik at ønskede tagger overgår grensene til Geo-område.

Interaktivt kart

Mye av applikasjonen kommer til å være basert rundt det interaktive, ved bruk av tags og geo-område vil masse pins og slikt dukke opp på kartet som du kan trykke deg inn på for detaljer, slik som vist på bildet.
